# mod_bauhaus  Bauhaus風パターン

バウハウスは、ドイツの歴史的美術学校の名前であると同時に、その流れを汲んだ「モダン」な機能主義的製品デザインを示します。
建築デザインの領域だけではなく、テキスタイルにおいても幾何学的なタイルを整然と配置したパターンをバウハウス風と呼ぶようです。

本モジュールでは、シンプルなタイルを敷き詰める形でバウハウス風パターンを生成します。なお、タイルの形状マスクは後述のようなテンプレートで追加可能になっています

## Wallpaper本体での設定

- タイル表示基本色として色1,色2 および 色1、色2のバリエーションに用いる明度加算値を指定します
  - タイルパターンの色として、色1、色2、色1+明度加算、色2+明度加算の4色を用います
- 背景色は単色になりますが、HOLD機能で別モジュールで生成した背景と合成できます
- パターン幅でタイルの大きさを指定します(タイルは正方形になります)

## 拡張設定画面

- プレビュー画面をクリックすると拡張設定画面が開きます
- 拡張設定画面では、タイルパターンごとの利用、反転回転、生成確率を設定します
  - チェックボックス： チェックを入れたパターンが生成時に利用されます
  - フラグに r を指定しているものは元図形の図形と地を反転したパターンも利用されます
  - フラグに数字を添えたものは、回転パターンの生成を数字の数に制限します(添字無しなら4方向)
  - 生成確率は任意の実数です。内部では表示対象の全確率の総和で正規化するので、相対的な重み付けになります
    - パターンaとパターンbを利用する時、確率がa-3.0, b=1.0になっていれば、内部ではa=0.75、b=0.25で扱われます
- resetボタンを押下すると、全パターンの設定を初期設定値に戻します

## パターンの追加

パターン生成は以下のテンプレートで生成します

```
@register
def mask_script(x,y,S):
	<生成コード>
	return <(S,S)のnumpy配列>
```

マスクはnumpy配列で生成します。円マスクであれば下の例のようになります。

```
@register
def circle(x, y, S)
    r = S // 2:
	return ((x-r)**2 + (y-r)**2 <= r**2)
```

マスク関数を作成したら、グローバル辺陬のDEFAULT_WEIGHTSに重み登録します。
円なので回転対称のため、フラグは r1 もしくは n1 とします。

```
   ["circle",      'n1', 0.1],
```

※ 確率を9,9などとするとその形状が大量に配置されるので試しで形状の確認をしたい場合に便利です
